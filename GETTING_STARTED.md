# Getting Started

## requirements

* git 
* nodejs v.18

## steps

1. clone project
    > git clone https://github.com/bsorrentino/copilot-cli-agent.git
2. npm install
3. npm run build
4. set environment variables
 * `COMMANDS_PATH`=<project path>/commands
 * `OPENAI_API_KEY`=<openai key>
5. npm run start:simple
